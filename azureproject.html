STEP 3

Now, what I wanted to do was to transform data. In order to do so I used databricks.

I started by creating cluster, which is simply a computational resource needed for data transformations. I chose the cheapest available option, since I didn't have lots of data to process.

Having my cluster created, I could finally write some code in the notebooks. I started with credential passthrough, which I have taken from Microsoft documentation and filled with my custom information.

While writing the code I encountered an error "This request is not authorized to perform this operation using this permission." And the solution turned out to be to assign the Storage Blob Data Contributor role to this service. It was quite surprising to me, taking into account that I'm a owner... But I guess that's just how it is...
